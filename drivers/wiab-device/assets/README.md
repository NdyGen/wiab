# Device Assets

This directory contains the device icons for the WIAB occupancy sensor.

## Required Files

You need to create the following image files:

### Icon (SVG)
- **File**: `icon.svg`
- **Size**: Scalable vector graphic
- **Description**: Main device icon shown in Homey interface
- **Suggested design**: A wasp icon inside a box/container outline

### Device Images (PNG)
Place these in the `images/` subdirectory:

- **small.png** - 75x75 pixels - Shown in device lists
- **large.png** - 500x500 pixels - Shown during pairing
- **xlarge.png** (optional) - 1000x1000 pixels - High-res version

## Design Guidelines

The icon should visually represent:
- A wasp (representing motion/presence)
- A box/container (representing the room/zone)
- Occupancy detection concept

**Suggested colors**: Orange (#FF6600) to match the app brand color

## Creating Icons

You can use tools like:
- **Figma** or **Adobe Illustrator** for SVG
- **Photoshop** or **GIMP** for PNG files
- Online icon generators like **Canva** or **Flaticon**

## Placeholder

Until you create custom icons, you can use simple placeholder images. The app will work without custom icons, but they improve the user experience.
