Define States

Define your room states as a JSON array. Each state can have transitions based on zone activity.

JSON array of state configurations. See example below.
Example configuration:
[
  {
    "id": "unoccupied",
    "name": "Unoccupied",
    "activeTransitions": [
      {"targetState": "occupied", "afterMinutes": 0}
    ],
    "inactiveTransitions": []
  },
  {
    "id": "occupied",
    "name": "Occupied",
    "activeTransitions": [
      {"targetState": "working", "afterMinutes": 15}
    ],
    "inactiveTransitions": [
      {"targetState": "unoccupied", "afterMinutes": 5}
    ]
  },
  {
    "id": "working",
    "name": "Working",
    "parent": "occupied",
    "activeTransitions": [],
    "inactiveTransitions": [
      {"targetState": "unoccupied", "afterMinutes": 10}
    ]
  }
]
State ID to use when device initializes
