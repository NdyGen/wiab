Intelligente Raumbelegung, die auch dann aktiv bleibt, wenn Sie still sitzen.

DAS PROBLEM
-----------
Das Licht geht aus, während Sie lesen. Die Heizung schaltet ab, während Sie arbeiten.
Bewegungsmelder erkennen nur Bewegung - keine tatsächliche Raumbelegung.

DIE LÖSUNG
----------
Kombinieren Sie Bewegungsmelder mit Türkontakten:
- Bewegung erkannt → Raum belegt
- Stundenlang still sitzen → Immer noch belegt
- Tür öffnet → Raum leer

Perfekt für Büros, Wohnzimmer und Schlafzimmer, wo Menschen sitzen, lesen oder schlafen.

VERWENDUNG
----------
1. Fügen Sie ein WIAB-Gerät hinzu und wählen Sie beim Koppeln Ihre Bewegungsmelder und Türkontakte aus
2. Fügen Sie das WIAB-Gerät zu Ihrem Raum hinzu
3. Schließen Sie die einzelnen PIR-Sensoren von der Raum-Aktivitätsberechnung aus
4. Lassen Sie WIAB den Belegungsstatus des Raumes bestimmen

Ergebnis: Homeys Raum-Aktivität wird zuverlässiger - kein falsches "Raum leer" mehr,
wenn Sie still sitzen!
